# Animal-Personality-Quiz

## Overview
This project is a short quiz that tells you what animal you are based on your personality. 

## Features 
- 10 questions
- Tracks with a scoring system
- Tells you your animal

## Possible Results
- Cat
- Dog
- Fox
- Owl

## How to Use
1. Answer all questions
2. Click "See Result"
3. View your personality result

## Languages
- HTML
- CSS
- JavaScript
